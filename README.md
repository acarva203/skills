## agent-skills
AI Agent skills I've built for specific use cases and repeatable tasks

## Reference
Skills split on one axis — who can invoke them. User-invoked skills are reachable only when you type them (e.g. /deadlines). Model-invoked skills can be reached by the agent automatically when the task fits.
