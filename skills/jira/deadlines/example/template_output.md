### Priority Key

| Icon | Priority | Use when… |
|------|----------|-----------|
| 🔴 | Critical | Blocker; production down or release-blocking |
| 🟠 | High | Must ship this sprint |
| 🟡 | Medium | Planned for this or next sprint |
| 🟢 | Low | Nice to have; no hard deadline |

### Status Key

| Status | Meaning |
|--------|---------|
| To Do | Not started |
| In Progress | Actively being worked on |
| In Review | PR open / awaiting approval |
| Blocked | Waiting on dependency or decision |
| Done | Complete and merged |

### Tips
- **Days Remaining** — bold values ≤ 5 to make near-deadline tickets stand out at a glance.
- For team view, group rows by **Assignee** as an alternative to sorting by date.
- Add a **Notes** column for blockers or dependency links if needed.
